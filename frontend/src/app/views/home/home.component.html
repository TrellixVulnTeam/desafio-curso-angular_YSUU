Bem-Vindo! Sistema para exemplificar a construção de uma aplicação de cadastro em Angular.

Este sistema é capaz de:
  • Visulizar a lista de produtos cadastrados
  • Criar novos produtos
  • Editar produtos existentes
  • Deletar produtos da lista

Visulizar produtos

  1. Clique na aba Produtos da barra lateral
  2. Agora você pode ver uma lista com todo os produtos

Criar novos produtos

  1. Clique na aba Produtos da barra lateral
  2. Clique no botão de "Novo Produto"
  3. Preencha as informações do produto que deseja criar
  4. Clique no botão "Salvar"

Editar produtos existentes

  1. Clique na aba Produtos da barra lateral
  2. Clique no ícone do lápis correspondente ao produto que deseja editar
  3. Altere as informações do produto
  4. Clique no botão "Alterar"

Excluir produtos da lista

  1. Clique na aba Produtos da barra lateral
  2. Clique no ícone da lixeira correspondente ao produto que deseja editar
  3. Clique no botão "Excluir"
